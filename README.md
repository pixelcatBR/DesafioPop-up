# Desafio do Pop-up

Este é um código py que fiz para um vídeo no youtube. Ele cria pop-ups na tela de tempos em tempos e usa subprocess.Popen para acumular.

## como usar

- primeiro modifique o time.sleep para o tempo desejado
- digite esse comando no seu terminal: ```python3 main.py```
